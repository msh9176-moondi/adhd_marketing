CHAPTER 06

지루함, 도파민,
보상 루프

  • 휴대폰을 확인하고 싶다
  • 갑자기 검색을 하고 싶다
  • 커피를 마시고 싶다
  • 새로운 아이디어가 떠올라 하던 일을 밀어낸다
"나는 왜 보상이 없으면 움직이지 못하지?"
보상은 단순한 유혹이 아니다. 제대로 설계하면 실행을 반복하게 만드는 강력한 루프가 된다.
01 ADHD에게 지루함은 단순한 싫증이 아니다

지루함 = 보상 회로 저작동 상태

지루함을 태도 문제로 보면 해결책은 "참아라"뿐이다. 보상 회로의 문제로 보면 설계할 수 있다.

핵심 개념

ADHD 성향을 가진 사람은 보상 회로가 저작동할 때 쉽게 지루해지고, 보상을 갈망하며, 자극이나 흥분을 추구한다. 이것은 의지의 문제가 아니라 뇌 작동 방식이다.

❌ 기존 관점
지루함 = 태도 문제 → 해결책: "참아라", "집중해라"
✓ 새로운 관점
지루함 = 보상 회로 문제 → 해결책: 작은 자극과 작은 보상을 붙인다
지루한 과제 → 시작 가능하게 바꾸기
서류 정리하기 → 서류 3장만 분류하고 체크 표시하기
보고서 작성하기 → 첫 문단 제목만 적기
공부하기 → 문제 2개 풀고 동그라미 치기
02 도파민은 쾌락보다 '움직이게 하는 힘'에 가깝다

도파민은 '기분 좋다'가 아니라
'해볼 만하다'는 감각이다

도파민은 흔히 쾌락 물질로 알려져 있지만, ADHD 실행에서 도파민을 쾌락으로만 보면 부족하다. 도파민은 행동을 시작하게 만드는 힘과 연결된다.

도파민의 실제 역할
도파민 증가
→
전전두피질 자극
→
계획 · 분류 · 순서 · 기억 · 평가 기능 활성화

먼 미래 보상의 문제점

  • "3개월 뒤에 좋아질 거야"
  • "언젠가 도움이 될 거야"
  • "나중에 큰 성과가 있을 거야"
머리로는 이해된다. 하지만 지금 당장의 실행력을 자극하기에는 충분하지 않다. 필요한 것은 행동 직후에 따라오는 작은 피드백이다.

즉각적인 작은 피드백 예시

  • 📖책을 한 장 읽고 표시하기
  • 👕운동복을 입고 체크하기
  • ✉️메일함을 열고 완료 칸에 동그라미 치기
  • 🎵10분 작업 뒤 음악 한 곡 듣기
03 보상은 유혹이 아니라 배치의 문제다

실행 전 보상 = 회피
실행 후 보상 = 강화

보상 자체가 문제가 아니다. 보상이 어디에 놓이느냐가 결과를 바꾼다. 같은 보상이라도 위치에 따라 회피가 되거나 강화가 된다.

핵심 원칙
실행 전에 보상이 오면 회피가 된다.
실행 후에 보상이 오면 강화가 된다.
❌ 실행 전 보상 (회피) ✓ 실행 후 보상 (강화)
공부 전 영상 보기
10분 공부 후 짧은 영상
정리 전 간식 먹기
물건 5개 정리 후 간식
시작 전 SNS 확인
10분 집중 후 SNS 5분

좋은 보상의 조건

  • ✓
    시간이 적절하다 (너무 길지 않다)
  • ✓
    끝이 분명하다
  • ✓
    다음 행동으로 돌아올 수 있다
  • ✓
    실행 뒤에 온다
04 작은 보상은 습관을 만든다

뇌에 '다시 시도할 이유'를
남기는 것

필요한 보상은 크고 거창한 것이 아니다. 오히려 너무 강한 보상은 다음 행동을 삼켜버린다. 중요한 것은 작고, 빠르고, 분명한 보상이다.

✅체크 표시 하나
⭐스티커 하나
☕커피 한 모금
🚶짧은 산책
🎵좋아하는 음악 한 곡
📝"했다"는 기록
1
작은 행동을 한다운동복 입기, 책 한 문장 읽기, 파일 열기
2
작은 보상을 받는다체크 표시, 커피 한 모금, 음악 한 곡
3
뇌가 다시 시도할 이유를 얻는다"이 행동은 다시 해볼 만하다"는 신호가 남는다
4
반복하면서 실행 회로가 조금씩 강해진다작은 성공이 쌓이며 자기 신뢰가 만들어진다
실전 예시 · 운동 루틴 만들기
1️⃣ "운동하기" → 운동복 입기 + 체크
2️⃣ 다음 날 → 운동복 입고 문밖으로 나가기
3️⃣ 그다음 → 5분 걷기

핵심: 한 번에 완벽한 루틴이 아니라, 행동과 보상을 연결하는 회로를 만드는 것이다.

"보상은 마지막에 주어지는 상이 아니다. 보상은 다음 행동을 가능하게 만드는 연결 고리이자 마중물이다."
6장 핵심 요약
  • 1 지루함은 태도 문제가 아니라 보상 회로 저작동 상태다. 지루한 일도 작게 만들고 완료 신호를 주면 시작할 수 있다.
  • 2 도파민은 쾌락이 아니라 '해볼 만하다'는 감각이다. 먼 미래 보상보다 지금 당장의 작은 피드백이 실행력을 자극한다.
  • 3 보상의 위치가 핵심이다 — 실행 전은 회피, 실행 후는 강화. 보상을 금지하지 말고 배치를 바꿔라.
  • 4 작고 즉각적인 보상이 뇌에 '다시 시도할 이유'를 남기고, 반복되면 실행 회로가 조금씩 강해진다.
