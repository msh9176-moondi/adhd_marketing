🔗 Google Sheets 연동

Apps Script 배포 URL을 입력하면 피코치·코치가 제출한 데이터가 실시간으로 Sheets에 저장되고, 아래 버튼으로 불러올 수 있습니다.

Apps Script 설정 순서
  1. Google Sheets 새 파일 생성
  2. 상단 메뉴 → 확장 프로그램 → Apps Script
  3. Code.gs 파일 내용을 복사해서 붙여넣기
  4. 저장(Ctrl+S) → 배포 → 새 배포
  5. 유형: 웹 앱 / 액세스: 모든 사용자 선택 → 배포
  6. 생성된 URL을 위 입력란에 붙여넣기
